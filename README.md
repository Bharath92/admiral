# Admiral - the Shippable installer

## CLI

```
  Usage:
    $./admiral.sh <command> [flags]

  Examples:
    $0 install --help
    $0 install
    $0 install --release v5.2.1     //Install on localhost with 'v5.2.1' release

  Commmands:
    install         Run Shippable installation
    help            Print this message
```
